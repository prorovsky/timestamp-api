Timestamp API

For that api you can provide human readable date like January 20 2020 and receive back unix time.

Or provide unix time and get back human readable date.

Example usage

http://prorovsky-timestamp-api.herokuapp.com/January-20-2020

http://prorovsky-timestamp-api.herokuapp.com/1579467600

Example output

{ "unix": "1579467600", "natural": "January 20, 2020" }

Or

{ "unix": null, "natural": null }

If you provided incorrect date.
